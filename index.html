Waze Drives Converter Tool

This applet will take a file you select with drives from the Waze personal data archive (which you can export from waze) and convert it into a GPX file showing your routes. See this link for more information about this data.
The idea to convert the data into GPX is taken from this waze discussion post
This software does not upload your data to the server. It just converts it in the memory of your web browser.
